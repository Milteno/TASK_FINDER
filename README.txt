Serwis do Szukania Pracy Dorywczej

Opis Projektu

Technologie
Frontend - 
React.js: Biblioteka JavaScript do budowania interfejsów użytkownika.
Redux Toolkit: Narzędzie do zarządzania stanem aplikacji.
React Router: Biblioteka do zarządzania routingiem w aplikacjach React.
Backend - 
Node.js: Środowisko wykonawcze JavaScript po stronie serwera.
Express.js: Framework webowy dla Node.js, który upraszcza tworzenie serwerów HTTP.
MongoDB: Nierelacyjna baza danych typu NoSQL przechowująca dane w formacie dokumentów BSON.
JWT (JSON Web Token): Technologia uwierzytelniania użytkowników i bezpiecznego przesyłania informacji między klientem a serwerem.

Wymagania:
Node.js: wersja 14.x lub wyższa
MongoDB: wersja 4.x lub wyższa

Instrukcja startu projektu:

Używając terminala przejdź do projektu
"cd task-finder/my-app"
Wykonaj polecenie npm install, będąc w tym katalogu w terminalu, aby zainstalować pakiety i biblioteki
"npm install"
Bedąc w katalogu my-app, aby uruchomić node.js użyj polecenia"
"node server.js"
